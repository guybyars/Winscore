
WINSCORE Soaring contest scoring program.

This GitHub repository contains the source code and related files for the Winscore contest scoring program developed and maintained by Guy Byars.

Some notes about this repository.

 - The source code can be accessed and build by anyone with an interest in such things.
 
 - The code and programs are only to be used for personal education and the advancement of soaring contest scoring.  NOT for any commerical projects.
 
 - Collaboration on this work is possible.  Contact Guy Byars to coordinate.
